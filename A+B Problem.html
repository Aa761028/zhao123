A+B Problem

时间限制: 1s | 内存限制: 128MB

题目描述

给定两个整数 a 和 b，求它们的和。

这是一道经典的入门题目，几乎所有的编程语言都可以轻松解决。

输入格式

输入包含两个整数 a 和 b，用空格分隔。

输出格式

输出一个整数，表示 a 和 b 的和。

样例输入

1 2

样例输出

3

数据范围

-10⁹ ≤ a, b ≤ 10⁹

提示

注意处理可能的整数溢出问题。

不同编程语言的输入输出方式：

  • C/C++: 使用 scanf/printf 或 cin/cout
  • Java: 使用 Scanner 类
  • Python: 使用 input() 函数
